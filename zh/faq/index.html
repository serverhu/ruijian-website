FAQ — 关于产品和服务的常见疑问

首页/常见问题
瑞建石油装备主要生产哪些产品？
我们主要生产 API 5CT 标准的石油套管（114.3-273mm）、油管（48.3-114.3mm）、钻杆、钻铤、无缝钢管、接箍及短节等石油专用管材产品。钢级覆盖 J55、N80、L80、P110 及 L80-13Cr 不锈钢。
产品是否通过 API 认证？
是的，我们的产品严格按照 API 5CT 标准生产，公司已通过 ISO 9001、ISO 14001、ISO 45001 等管理体系认证，并持有相关资质证书。
年产能是多少？
油套管年产能 15 万吨，加厚油管年产能 3 万吨，接箍年产能 50 万支。拥有 4 条自动化生产线和 1 条加厚/接箍智能线。
是否接受非标定制？
可以。我们提供定制化服务，可根据客户需求生产非标规格、特殊扣型的产品。请联系我们的销售团队提供具体需求。
最小起订量（MOQ）是多少？
常规产品的最小起订量一般为 50-100 吨，具体取决于规格和钢级。定制产品的起订量需根据工艺评估后确定。详情请咨询我们的销售团队。
产品可以出口吗？支持哪些付款方式？
可以。我们通过天津销售中心开展全球出口业务，产品已出口至中东、非洲、中亚等地区。支持 T/T、L/C 等国际贸易付款方式。
工厂和销售中心分别在什么位置？
制造基地位于黑龙江省双鸭山市岭东区双选路钢联；销售中心位于天津市东丽区军粮城工业园（瑞建石油装备（天津）有限公司）。
